# 弹幕组件分享源码
在线演示地址：https://duooduo.github.io/share02/zan-oop.html
 
## 实现效果
1. 弹幕循环滚动效果，点击之后获得点赞效果并冒桃心。
2. 桃心随自身位置不断变浅，超出阀值隐藏。
3. 弹幕随自身滚动的相对位置改变透明度，超出阀值隐藏。
4. 利用对象封装，使两个弹幕区域互不影响。
